

youtube-dl is a Python based small command-line tool that allows to download videos from :
1. YouTube.com
2. Dailymotion
3. Google Video
4. Photobucket
5. Facebook
6. Yahoo
7. Metacafe
8. Depositfiles 


It written in pygtk and requires Python interpreter to run this program, it’s not platform restricted. It should run on any Unix, Windows or in Mac OS X based systems.
